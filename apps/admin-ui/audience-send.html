名單發送（群組 / 頻道）

使用說明

輸入 User ID 後，可以載入此帳號可見的群組或頻道，再讀取成員並發送訊息。

此功能會依照目前 TDLib session 執行，請確認該帳號具備讀取成員與發送訊息的權限。

選擇群組與成員

請輸入已建立 TDLib session 的 user_id。
可手動輸入，或從左側群組清單選擇。
每頁最多 200 位成員，預設讀取 10 頁。
