名單發送（群組 / 頻道）

操作說明

先輸入登入後的使用者 ID，載入可用的群組 / 頻道清單，再選一個來源抓取 members 名單。
目前這頁的核心流程是：選來源 → 載入 members → 準備發送。
這裡的群組 / 頻道只是 名單來源，不是直接把訊息發到群組裡。
如果後端的 POST /supergroups/{user_id}/{chat_id}/send 尚未完成，這頁的發送按鈕會先提醒你。
